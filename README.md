# Authentication System (Vanilla JavaScript)

A simple authentication system built using HTML, CSS, and JavaScript.  
This project demonstrates user login, registration, role-based access, password validation, and data persistence using localStorage.

---

## Features

- User login system
- User roles (Admin / User)
- Password validation with strength rules
- Account lock after 5 failed login attempts (5 minutes)
- Show / hide password functionality
- Change password feature
- Admin dashboard to view all users
- Data persistence using localStorage
- Predefined default users

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
- Browser localStorage API

## How It Works

- Users can log in using username and password
- Credentials are checked against stored users
- After 5 failed attempts, account is locked for 5 minutes
- Admin users can view all registered users
- Users can change their password if authenticated
- All data is stored in localStorage

---

## Password Requirements

A valid password must contain:

- At least 8 characters
- One uppercase letter
- One lowercase letter
- One number
- One special character (@$!%*?&)
